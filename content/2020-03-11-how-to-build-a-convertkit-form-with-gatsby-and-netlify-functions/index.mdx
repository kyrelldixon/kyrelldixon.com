If you've worked with Gatsby, you've probably heard of Netlify. Netlify is a tool that makes it easy to deploy static sites of all kinds. Not only that, but they give you access to AWS Lambda which lets you write server logic without managing a server.

This tutorial will show you exactly what you need to know to set them up together.

## Get Netlify Dev

Netlify Dev is an open source tool that helps developers test their deployments to Netlify. In this case, you'll need it to locally test the the functions you will later deploy to Netlify.

To install it, use:

```bash
yarn global add netlify
```

I'll be using `yarn` as my package manager throughout the tutorial, but you can also use `npm` to install packages.
